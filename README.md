# ConfuAC
Lightweight cross-platform anti-cheat for Minecraft 1.8.8+. Detects autoclicker, reach, speed, fly, scaffold, anti-KB, and aimbot. Supports Spigot, Velocity &amp; BungeeCord with real-time cross-server alerts. LuckPerms integration, built-in mute/ban system, configurable thresholds. Easy to use, highly customizable.
